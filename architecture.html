GPU-Aware Model Routing

HALO Core intelligently routes requests to the optimal AI model based on task context and workspace.

Workspace Model GPU Use Case
General 7B Coordinator GPU 1 Fast responses, simple queries
Coding 32B Coder GPU 0 Code generation, analysis, debugging
Reasoning 32B R1 GPU 0 Complex logic, multi-step problems
Trading 32B Orchestrator GPU 0 Market analysis, trading decisions
Security 32B Orchestrator GPU 0 Vulnerability assessment, OSINT

GPU Hot-Swapping: Only one 32B model loaded at a time on GPU 0, automatically swapped based on demand. The 7B coordinator on GPU 1 remains always available for fast queries.
